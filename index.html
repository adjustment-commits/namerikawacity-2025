第1回：立つことから始める

— 足底の感覚が動きを変え、動きが将来につながる —

この講習は「フォームをその場で直す場」ではなく、良い選手の体がどんな構造で動いているかを知る場です。

今日だけですべてを身につける必要はありません。頭の片隅に「体の使い方の地図」を残すことが目的です。

トレーニングは通年で続けてこそ力になります。今日の学びは、そのスタート地点です。
講習の趣旨

この講習で目指すこと

トレーニングや動きづくりは、本来1年を通してコツコツ続けるものです。今日の講習だけで完璧にできるようにはなりません。

  • 良い選手の体はどういう構造でできているか
  • 良い選手の動きはどんな順番で起きているか
  • その一部を実際に体感してみること

この3つを知って体験しておけば、今後どこかのタイミングで「点」と「点」がつながる可能性が高くなります。

今日覚えきれなくても、印象に残った感覚や言葉がひとつでもあれば十分です。
実例

上のカテゴリーの選手は何が違う？

社会人ジャパン候補に選ばれるレベルの打者は、バッティングのとき、

  • 胸郭と肩甲帯の分離が当たり前のようにできている
  • 骨盤と下肢の連動で下半身からエネルギーを上に伝えている

だからこそ、上のカテゴリーでプレーできます。

ただし、そんな選手でも、完全にコントロールできているわけではありません。代表に定着したり、プロを目指したりするなら、さらに細かいトレーニングが必要です。

今日みんなに体験してもらうのは、「上手い選手が当たり前にやっている体の使い方」の入り口です。
ゴール

今日のゴール

  • 足の裏が、身体の「地図」になっていることを知る
  • 良い立ち方が、投球・打撃の土台になることを理解する
  • 軸が整うと、野球の動きが楽になることを体験する

フォームを変える前に、「立つ」を変える。これが第1回のテーマです。

考え方

なぜ「立つ」が最初なのか

野球のすべては、

地面 → 足 → 体 → ボール（バット）

という順番で力が伝わっています。

  • 立ち方が悪い＝力が途中で逃げる
  • フォームの前に「土台」を整える方が、結果が出やすい
立ち方が安定すると、上半身の力みが自然に減ります。
足底の役割

足底は「センサー」

足の裏には、

  • どこに体重がかかっているか（位置）
  • どれくらい圧がかかっているか（圧）
  • どの方向に揺れているか（揺れ）

を感じるセンサーがたくさんあります。

このセンサーが狂うと、膝・股関節・腰まで影響します。

基本

足底の三点

まずは足の裏の「三点」を意識します。

  • 母趾球（親指のつけ根あたり）
  • 小趾球（小指のつけ根あたり）
  • 踵（かかと）

この3点で「面」で地面を捉えると、体が安定します。

アーチ

アーチの3つの役割

足のアーチには、主に3つの役割があります。

  • ① 衝撃を吸収する
  • ② バランスを調整する
  • ③ 地面からの力を伝える（推進力）

アーチは、潰れすぎてもダメ、硬すぎてもダメです。

アーチ

3つのアーチ

  • 内側アーチ（親指側の土踏まず）
  • 外側アーチ（小指側）
  • 横アーチ（指のつけ根をつなぐライン）

特に横アーチが潰れると、母趾球で踏む感覚が消えやすくなります。

横アーチが潰れる → つま先側がペタッと広がる → 踏ん張りにくい足になります。
よくある例

悪い足底の例

  • 母趾球が使えず、外側ばかりで立っている
  • 小趾球が浮いている（外側アーチが使えていない）
  • 踵重心で、後ろに倒れやすい

これらは中学生にとても多いパターンです。

距骨

距骨とは？

距骨（きょこつ）は、足首の中心にある「方向を決める骨」です。

  • 距骨が固まる → 回内・回外が出ない
  • 結果として、立位が不安定になりやすい

足首の「自由度」は、距骨の動きにかなり影響を受けます。

重心

重心MAP（前後）

重心が前後にズレると、動きのクセが変わります。

  • 前すぎる → ブレーキが多くなる、突っ込みやすい
  • 後ろすぎる → 押し出しが弱くなる、出足が遅い

中学生は特に「後ろ重心」が多いです。

重心

重心MAP（左右）

左右の重心ズレも、プレーに影響します。

  • 右にズレる → 右に倒れやすい、左への動きが弱い
  • 左にズレる → 軸が作れず、回旋が不安定

投手では特に、「右に倒れる癖」が球質やコントロールを崩すことがあります。

チェック

良い立ち方の指標

  • 足底三点で、しっかり地面を捉えている
  • 膝が内・外にねじれすぎていない
  • 骨盤が前にも後ろにも倒れすぎていない
  • 胸郭が自然に広がり、肩に力みがない

難しければ、まずは「三点で立つ」ことから始めます。

よくある例

悪い立ち方の例

  • 反り腰で、腰だけで立っている
  • 猫背で、母趾球が使えていない
  • 過度な内股／外股
  • 片脚でほとんど立てない

こうした立ち方は、投球・打撃の「フォームの崩れ」に直結します。

考え方

姿勢は「結果」

姿勢そのものは「結果」です。

原因の多くは、

  • 足底の使い方
  • アーチの状態
  • 距骨の動き

にあります。

上だけ直そうとするのではなく、「足」から整えていきます。

連鎖

足底 → 膝 → 骨盤 → 胸郭のつながり

立つだけで、

  • 膝の向き
  • 骨盤の傾き
  • 胸郭の角度

まで変わります。

動きを治す前に、「立つ」を整える方が効率が良いのはこのためです。

体験①

ここで体験：足指ワーク

実際に、足の指とアーチを動かしてみます。

  • 足指グーパー（指をしっかり動かす）
  • 母趾を軽く押して「踏む」感覚を確認
  • 横アーチの沈み込みを左右で比べる
足指が動きにくい選手は、立位の安定性が落ちていることが多いです。
体験②

ここで体験：重心の位置を変えて立つ

重心を意図的に動かして、感覚の違いを体験します。

  • 前重心で立つ
  • 中央で立つ
  • 後ろ重心で立つ

それぞれで、

  • どこが力を入れやすいか
  • どこで投げやすい／打ちやすいか

を感じてもらいます。

チェック

片脚立ちチェック

片脚で立つことで、「軸」の強さをチェックします。

  • 5秒以内に大きくグラつく
  • 軸が横に流れてしまう
  • 小趾球がまったく使えていない

こうした選手は、投球・打撃でも「軸ブレ」が起きやすくなります。

デモ

立つだけで変わる：デモ

立ち方を整える前後で、

  • 腕の上がりやすさ
  • 肩のチカラ感
  • 踏み込みの準備姿勢

がどう変わるかデモで見せます。

選手自身に「さっきと違う」と言わせられると、理解が一気に進みます。
野球動作

野球動作への接続①：投球

立ち方の違いが、投球にどう出るか。

  • 悪い例：後ろ重心 → ステップが弱い、ボールが抜けやすい
  • 悪い例：外側荷重 → ボールが引っかかったり、すっぽ抜けたりする
  • 良い例：母趾球が働く → ステップ方向が安定し、コントロールしやすい
野球動作

野球動作への接続②：打撃

立ち方の違いが、打撃にどう出るか。

  • 悪い例：踵重心 → 軸足が潰れ、ヘッドが遅れる
  • 悪い例：アーチが潰れる → トップが不安定で、スイング軌道がブレる
  • 良い例：足底三点 → トップが安定し、最短距離でヘッドが出やすい
軸

軸は「立つ」で決まる

軸がズレると、

  • 投げる方向
  • バットヘッドの軌道
  • 身体の回旋のタイミング

すべてがズレます。

軸を作る最初の一歩が「立ち方」です。

体験③

ここで体験：軸ブレドリル

片脚立ちで、グラブ側から軽く押してみます。

  • 軸が弱い選手 → 簡単に崩れる
  • 足底を整えた後 → 同じ力でも崩れにくくなる

足底の感覚が変わるだけで、軸の強さが変わることを体験させます。

体験④

ここで体験：バットなしトップ

バットを持たずに、トップの形だけを作ります。

  • 足底を整えてからトップを作る
  • 肩の軽さ・胸郭の動きやすさを確認
  • 骨盤がスムーズに使えるかチェック

「立ち方を変えるだけでトップの安定感が変わる」ことを実感させます。

まとめ

今日のまとめ：構造は残る

  • 足底は身体のセンサーであり、「三点」で立つことが基本
  • 良い立ち方が、投球・打撃の土台になる
  • 軸は「立つ」で決まり、フォーム以前に整える価値がある
  • 今日すべてをできなくても、「体がこう動く」という構造のイメージが残れば十分

トレーニングは通年で続けてこそ身につきますが、構造の理解と体験は一生もののベースになります。

今日の感覚をどこかで思い出せたとき、それが「点」と「点」がつながる瞬間になります。
