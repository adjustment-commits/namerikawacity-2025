第4回：肩甲帯と肋骨

テーマは「軸足が回っても手を出さない」。

自分の体とメディシンボールだけで、3時間やり切る。

前回の成果

lesson03の後

  • ロングティーで飛距離が伸びた
  • 初速アップの選手が増えた
  • 守備でも「間」が作りやすくなった

ここまでは順調。だから次の症状が出る。

問題

いま起きている症状

  • 軸足と一緒に手が出る
  • 打球方向が引っ張り一択になりやすい
  • 体の回転に手が乗りすぎる
整理

これは悪化ではない

順番が整って、軸足が回れるようになった。

その結果、手まで同時に出る選手が増える。

構造が進んだから見えてきたズレ。ここを直せば次に行ける。

仮説

原因（仮説）

  • 軸足回旋のタイミングに肩甲帯と上肢が同時発火
  • 肋骨の意識はあるが、肩甲帯が肋骨に貼り付きやすい
  • 結果、手が体の近くを通りすぎる
狙い

第4回のゴール

ゴールはこの2つ
① 肩甲帯が肋骨から立体的に離れる・近づく
② 肩甲帯の回旋とともに上肢を90度前方へ解放していく

手を速くする回ではない。手を出さずに回れる体を作る回。

イメージ

解決策のイメージ

軸足を回すとき、

バットは肩口のまま

振っていないのに、コンタクトが起こる。

これを作るために「肋骨と肩甲帯」の関係をトレーニングで習得する。

構造

肋骨は「板」じゃない

肋骨先行で起こるのは、

  • 板が前に進む、ではない
  • 球体が回旋しながら前に進むに近い

回旋と前進が同時に起こる。だから肩甲帯の付き方がズレると手が出る。

チェック

今日見るポイント

  • 軸足が回っても、手が出ない
  • 肩甲帯が前にすべるのではなく、肋骨に沿って動く
  • 上肢が体から距離を取れる

良い動きを気にしない。見るのはズレだけ。

準備

3時間の組み立て

  • Aモビリティ（多め）
  • Bスタビリティ（多め）
  • Cフィジカル本編（自重 + MB）
  • D現場での統合（MBで強めに）

A/Bの比率や時間配分は現場の空気で判断してOK。

MBルール

メディシンボールのルール

  • 重さは軽めでOK（スピード優先）
  • 狙いは「出力」より順番
  • 疲れてきたら精度優先に戻す
EX01

構造① → エクササイズ

EX01：肋骨の丸みに沿って「巻く・ほどく」（自重）
場所：屋内 / 雨天練習場
回数：左右 各6回 x 2
道具：なし
やること：肋骨に沿って体幹を小さく回旋 -> 元へ
やること：肩は力を抜いて、肋骨の丸みに沿って「巻く -> ほどく」を丁寧に。
見るポイント：肩が先に動かない / 腰で誤魔化さない / 息が止まらない
EX02

構造① → エクササイズ

EX02：肩甲帯の「離れる・近づく」（自重）
場所：屋内 / 雨天練習場
回数：10回 x 2
道具：なし
やること：両腕前方で押す -> すっと戻す
やること：肩甲帯を前に逃がすのではなく、肋骨の上でスライドさせる。
見るポイント：首がすくまない / 胸を張らない / 肘で誤魔化さない
EX03

構造② → エクササイズ

EX03：上肢を90度前方へ解放（自重）
場所：屋内 / 雨天練習場
回数：左右 各6回 x 2
道具：なし
やること：肩甲帯の回旋に合わせて前方へ解放
やること：手を前に出すのではなく、肩甲帯の回旋の結果として上肢が前に出る。
見るポイント：腕だけで出ない / 体幹が潰れない / 反対側の肩が上がらない
EX04

肋骨 + 肩甲帯 → MBへ

EX04：MBホールドで「手を出さずに回る」
場所：屋内 / 雨天練習場
回数：左右 各6回 x 2
道具：メディシンボール
やること：胸の前で保持 -> 軸足だけ回す -> まだ投げない
やること：回っているのにボールがその場に残る感覚を作る。
見るポイント：回すと同時に手が動くならアウト / 肩が先に飛び出すならアウト
EX05

MBで「最後に解放」

EX05：MBリリースは最後（置いて -> 回って -> 放す）
場所：雨天練習場（推奨）
回数：左右 各6投 x 2
道具：メディシンボール
やること：保持 -> 先に軸足回旋 -> 最後に放す
やること：放す瞬間だけ上肢を90度前方へ解放する。
見るポイント：最初から腕で投げない / 体の近くで処理しない / 押し出さない
EX06

手が近すぎる対策

EX06：MBを体から離して扱う（距離の練習）
場所：屋内 / 雨天練習場
回数：左右 各6回 x 2
道具：メディシンボール
やること：保持位置を少し遠く -> 回っても距離を保つ
やること：体から距離を取ったまま、肩甲帯が肋骨に沿って動く感覚を作る。
見るポイント：肘が後ろに引けない / 肩がすくまない / 腰で回さない
現象の判別

手が出る2パターン

  • タイプ1：軸足回旋と同時に手が発火（タイミング問題）
  • タイプ2：手が体の近くを通りすぎ（距離問題）

どっちもlesson04で処理できる。現場で分類して当てるだけ。

屋内メニュー

屋内（基本）

  • EX01：肋骨の丸みに沿って巻く・ほどく
  • EX02：肩甲帯の離れる・近づく
  • EX03：上肢90度前方への解放
  • EX04：MBホールドで手を出さずに回る

屋内は精度。雑にやるとゼロになるのでテンポより質。

雨天練習場メニュー

屋外（雨天練習場）

  • EX05：MBリリースは最後（置いて -> 回って -> 放す）
  • EX06：MBを体から離して扱う（距離の練習）
  • 強度を上げるなら「回数」ではなくスピード

動きは派手に。見るポイントは少なく。

3時間運用

現場での回し方

  • 最初はA/Bを多めにして体を整える
  • 途中からMBを増やして派手にする
  • 崩れたらA/Bに戻して立て直す

空気で判断。最終的に「手が出ない」を作れたら勝ち。

今日のチェック

今日はこれだけ見ればいい

  • 軸足が回っても手が出てない？
  • 肩甲帯が肋骨から離れたり近づいたりできた？
  • 上肢が前方へ解放できて距離を取れた？

良い動きを気にしない。ズレだけ拾ってtoughに整える。

まとめ

第4回まとめ

  • 手が出るのは進化の途中で起こる症状
  • 鍵は肋骨と肩甲帯の関係
  • 軸足は回っていい。手はまだ出さない
  • 自重 + MBだけで、十分作れる
