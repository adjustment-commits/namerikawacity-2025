第1回：立つことから始める

上のカテゴリーの選手は特別な才能ではなく、身体の構造を自然に使えるから強い。

今日はその入口を知り、体験する時間。

今日だけで完璧になる必要はない。

ただ、「体の使い方の構造」を頭の片隅に置いてほしい。

構造の理解は後で必ず噛み合う瞬間が来る。
講習の趣旨

この講習で目指すこと

  • 良い選手の体がどんな構造で動いているかを知る
  • その一部を体験し「感覚のヒント」を持つ
  • 今日の理解が将来つながる“種”になる

技術は忘れるが「構造」は残り続ける。

実例

上のカテゴリーの選手は何が違う？

社会人ジャパン候補の選手は

  • 胸郭と肩甲帯の分離が自然にできる
  • 骨盤と下半身の連動で動きにブレがない

しかし、それでも「完全にコントロールできている」わけではない。

今日の講習はその入口を知ることです。
ゴール

今日のゴール

  • 足の裏が「体の地図」だと知る
  • 良い立ち方＝野球動作の土台と理解する
  • 軸が整うと動きが変わるのを体験する
考え方

なぜ「立つ」から始めるのか

野球の力は、

地面 → 足 → 体 → ボール

の順に伝わる。

  • 立ち方が悪い＝力が途中で逃げる
  • フォーム修正より「土台」を整える方が速い
足底の役割

足底は「センサー」

足の裏は

  • どこに立っているか（位置）
  • どれだけ体重がかかるか（圧）
  • どちらに揺れているか（揺れ）

を感知する重要なセンサー。

ここが狂うと膝・股関節・腰まで波及する。

基本

足底の三点

  • 母趾球
  • 小趾球
  • 踵

三点で「面」を作ると立位が一気に安定する。

アーチ

アーチの3つの役割

  • 衝撃吸収
  • バランス調整
  • 推進力の伝達

潰れすぎても硬すぎてもパフォーマンスは落ちる。

アーチ

3つのアーチ

  • 内側アーチ
  • 外側アーチ
  • 横アーチ

横アーチが潰れると母趾球が働きにくくなる。

例

悪い足底の例

  • 外側ばかりに乗る
  • 小趾球が浮いている
  • 踵重心で後ろへ倒れやすい

中学生にもっとも多いパターン。

距骨

距骨とは？

足首の中心にある「方向を決める骨」。

  • 距骨が固い → 回内・回外が出ない
  • 結果として立位が不安定に
重心

重心MAP（前後）

  • 前すぎ → 突っ込みやすい
  • 後ろすぎ → 出足が遅い

中学生は「後ろ重心」が多い。

重心

重心MAP（左右）

  • 右にズレる → 右へ倒れやすい
  • 左にズレる → 軸が作りにくい

投手は特に「右に倒れる癖」が球質を乱す。

チェック

良い立ち方の指標

  • 三点でしっかり立つ
  • 膝がねじれすぎない
  • 骨盤が前後に倒れすぎない
  • 胸郭が自然に広がる
例

悪い立ち方の例

  • 反り腰
  • 猫背＋母趾球が使えない
  • 内股・外股
  • 片脚で立てない
考え方

姿勢は「結果」

姿勢そのものは「結果」。 原因は、

  • 足底
  • アーチ
  • 距骨の動き

ここにある。

連鎖

足底 → 膝 → 骨盤 → 胸郭

立つだけで、

  • 膝の向き
  • 骨盤の傾き
  • 胸郭の角度

が変わる。

体験①

足指ワーク

  • 足指グーパー
  • 母趾を押して「踏む」感覚
  • 横アーチの沈み込みチェック

足指が動かない＝立位が弱い。

体験②

重心位置を変えて立つ

前 → 中央 → 後ろ それぞれで、

  • 力の入りやすさ
  • 投げやすさ／打ちやすさ

を感じる。

チェック

片脚立ちチェック

  • 5秒で大きく揺れる
  • 軸が流れる
  • 小趾球が使えない

これは投打の「軸ブレ」に直結。

デモ

立つだけで変わる

立ち方を整えると、

  • 腕の上がりやすさ
  • 肩の軽さ
  • 踏み込み準備

が変わる。

投球

投球への接続

  • 悪い例：後ろ重心 → ステップ弱い
  • 悪い例：外側荷重 → すっぽ抜け
  • 良い例：母趾球が働く → コントロール安定
打撃

打撃への接続

  • 悪い例：踵重心 → 軸足潰れる
  • 悪い例：アーチ潰れ → ヘッド遅い
  • 良い例：三点で立つ → トップ安定
軸

軸は「立つ」で決まる

軸がズレると、

  • ボール方向
  • ヘッド軌道
  • 回旋タイミング

すべてがズレる。

体験③

軸ブレドリル

  • 片脚立ちで軽く押す
  • 三点で立つと崩れにくい

足底の変化だけで軸の強さが変わる。

体験④

バットなしトップ

  • 足を整えてからトップ
  • 肩が軽くなる
  • 胸郭が動く

立ち方次第でトップの質が変わる。
